What is the Document Object Model (DOM)?

the structure of an HTML or XML document as a tree of objects, where each node in the tree is a part of the document (e.g., an element, attribute, or text). The DOM allows developers to interact with and manipulate the structure, content, and style of a web page using programming languages like JavaScript.

How do you select an element form the dom?

When you want to interact with a specific element on a webpage (e.g., to style it, change its content, or add event listeners), you first need to select it from the DOM.
